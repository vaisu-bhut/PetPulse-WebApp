# PetPulse Web App

## Overview
State-of-the-art interface for the PetPulse autonomous monitoring system. Built with Next.js 16, React 19, and Tailwind CSS v4.

## Features
- **Real-time Dashboard**: Live status of pet comfort and system health.
- **Alert Center**: Rich notifications with AI-analyzed context.
- **Video Library**: Archive of analyzed clips (Coming Soon).
- **Agent Settings**: Configure intervention thresholds (Coming Soon).

## getting Started

### Prerequisites
- Node.js 18+ (20+ recommended)

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
npm start
```
